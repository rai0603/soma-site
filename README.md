# Soma — 行銷網站

> **Soma — Agent in the Shell.**
> Bring your own AI; Soma gives it a face, a voice, and a memory.

Soma 桌面虛擬夥伴（Live2D / VRM）的官方行銷站。靜態網站 + Paddle.js Overlay Checkout。

- App 主 repo（私有）：`~/Developer/live2d-desktop-agent/`
- 授權後端（私有）：`~/Developer/soma-license/`（Paddle webhook + 序號 + 月費代管）
- 本站：公開、利 SEO；購買走 Paddle（買斷／月費／加購點數／外觀商店）

## 站型
群眾募資（群募）框架：募資儀表板 + 早鳥倒數 + 限量批次 + Stretch Goals；結帳走 Paddle Overlay。
